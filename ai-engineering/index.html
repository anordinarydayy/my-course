AI ENGBuilding on Foundation Models · Interactive Course

AI Engineering — The Full Course

Forty-eight self-contained, interactive lessons on building applications with foundation models — the AI layer that sits on top of strong software engineering. Organised by the nine areas of the field and walked in a dependency-aware order: foundations and prompting first, evaluation early, then RAG, agents, and the production concerns. Every lesson runs the same way — a clear mental model, schematic play-through diagrams, worked examples, and a self-test.

Original study material. Built from scratch for software engineers moving into AI work — the assumed starting point is solid DSA and system-design fundamentals. The framing draws on widely-discussed industry sources (Chip Huyen's AI Engineering, Anthropic's Building Effective Agents, and current provider documentation), but every explanation, diagram, and line of code is original. Pricing and model details are current as of mid-2026 and move fast — treated as illustrative throughout.
How to use it

Do L00–L02 first — the landscape, how the model works, and tokens/cost underpin everything else. Then walk the phases in order; evaluation (P2) comes early on purpose so every later phase has a way to tell whether a change helped. The guiding rule throughout: climb the adaptation ladder slowly (prompt → RAG → agents → fine-tune) and reach for complexity only when a simpler approach has failed you for a reason you can name.
