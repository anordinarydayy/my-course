DSAPattern-First · Interactive Course

Coding Interview DSA — The Full Course

Forty-two self-contained, interactive lessons organised by the ~20 patterns that actually recur in live-coding rounds. Every pattern lesson runs the same chain — brute force → the insight that removes the redundant work → the invariant that proves it — with schematic play-through diagrams, a fully worked LeetCode problem per technique, a graded practice ladder, and a self-test.

Original study material. Built from scratch for coding-interview prep (the live-coding loops common for SWE roles in Singapore: Grab, Sea/Shopee, GovTech, ByteDance/TikTok, the big-tech offices, and the banks). Problem names are well-known identifiers for anchoring only — every description, diagram, and line of code is original. Fifteen lessons are ★ Core — master those and you can solve the bulk of LeetCode medium. See the roadmap for the study plan, the ~40-minute mock protocol, and the self-grade rubric.

How to use it

Do L00–L01 first (the framework and complexity analysis underpin everything), skim Phase 1 as a refresher, then run the ★ Core 15 before the rest. Learn each invariant, not a memorised solution — then drill 6–10 tagged problems per Core pattern, timed, against the mock rubric.
